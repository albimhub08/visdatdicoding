# Proyek Analisis Data - E-Commerce Seller

## Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis persebaran seller berdasarkan state dan kota menggunakan E-Commerce Public Dataset.

## Pertanyaan Bisnis
1. Bagaimana persebaran seller di setiap state dan wilayah mana yang menunjukkan dominasi aktivitas penjualan?
2. Bagaimana persebaran seller di tingkat kota dan kota mana yang berpotensi menjadi pusat pertumbuhan bisnis?

## Insight
Hasil analisis menunjukkan bahwa seller terkonsentrasi pada wilayah tertentu, terutama state SP dan kota Sao Paulo. Terdapat ketimpangan distribusi seller yang membuka peluang ekspansi ke wilayah dengan jumlah seller yang lebih rendah.
